### [1] [Omics Data Discovery Agents](https://arxiv.org/abs/2603.10161)
*Alexandre Hutton,Jesse G. Meyer*

Main category: q-bio.GN

TL;DR: 提出一个基于LLM智能体的框架，自动获取组学相关文献并将非结构化信息转换为可搜索的研究对象，实现生物医学文献中数据的自动化大规模重用。


<details>
  <summary>Details</summary>
Motivation: 生物医学文献包含大量组学研究数据，但大多数已发表数据在计算上无法重用。原始数据虽然存放在公共存储库，但重现结果所需的关键信息分散在正文、补充文件和代码库中，中间数据位置不规律，导致数据难以访问和重用。

Method: 开发基于大语言模型（LLM）的智能体框架，配备工具集用于：获取组学研究、提取文章元数据、识别和下载已发表数据、执行容器化定量分析流程、运行分析以解决新问题。使用模型上下文协议（MCP）服务器暴露容器化分析工具。

Result: 1) 从PubMed Central文章自动提取元数据，标准数据存储库的数据集识别精度达80%；2) 智能体能识别相关文章、下载相关数据集并重新定量蛋白质组学数据，与报告预处理方法匹配时差异表达蛋白重叠率达63%；3) 智能体能识别语义相似研究、确定数据兼容性并进行跨研究比较，揭示肝纤维化中一致的蛋白调控模式。

Conclusion: 该工作为将静态生物医学文献转换为可执行、可查询的资源奠定了基础，实现了生物医学数据的大规模自动化重用，解决了文献数据难以访问和计算重用的关键问题。

Abstract: The biomedical literature contains a vast collection of omics studies, yet most published data remain functionally inaccessible for computational reuse. When raw data are deposited in public repositories, essential information for reproducing reported results is dispersed across main text, supplementary files, and code repositories. In rarer instances where intermediate data is made available (e.g. protein abundance files), its location is irregular. In this article, we present an agentic framework that fetches omics-related articles and transforms the unstructured information into searchable research objects. Our system employs large language model (LLM) agents with access to tools for fetching omics studies, extracting article metadata, identifying and downloading published data, executing containerized quantification pipelines, and running analyses to address novel question. We demonstrate automated metadata extraction from PubMed Central articles, achieving 80% precision for dataset identification from standard data repositories. Using model context protocol (MCP) servers to expose containerized analysis tools, our set of agents were able to identify a set of relevant articles, download the associated datasets, and re-quantify the proteomics data. The results had a 63% overlap in differentially expressed proteins when matching reported preprocessing methods. Furthermore, we show that agents can identify semantically similar studies, determine data compatibility, and perform cross-study comparisons, revealing consistent protein regulation patterns in liver fibrosis. This work establishes a foundation for converting the static biomedical literature into an executable, queryable resource that enables automated data reuse at scale.

Method: 分析2020-2023年五个COVID-19阶段14,181名18-24岁美国青年的47个自我报告心理健康症状数据，使用高斯图模型、症状社区识别和最小支配集模块控制分析

Abstract: The COVID-19 pandemic exposed young people to a prolonged and evolving societal stressor, yet it remains unclear whether symptom networks were reorganized or whether control was redistributed across a conserved modular scaffold. For each phase, we estimated Gaussian graphical models, identified symptom communities, and characterized minimum-dominating-set-based module control. Symptom networks showed broadly conserved community organization across phases, indicating a stable mesoscale scaffold despite marked temporal variation.
