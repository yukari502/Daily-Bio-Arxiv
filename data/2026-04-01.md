### [2] [ParetoEnsembles.jl: A Julia Package for Multiobjective Parameter Estimation Using Pareto Optimal Ensemble Techniques](https://arxiv.org/abs/2603.29986)
*Jeffrey D. Varner*

Main category: q-bio.QM

TL;DR: ParetoEnsembles.jl是一个Julia包，使用Pareto最优集成技术生成参数集成，无需梯度信息，改进了算法效率并支持并行计算。


<details>
  <summary>Details</summary>
Motivation: 在自然和人工系统的数学模型中，通常有许多可调参数需要从多个可能冲突的数据集中估计。与其报告单一的最佳拟合参数向量，生成参数集成集合能更好地映射竞争目标之间的权衡关系。

Method: 使用Pareto最优集成技术（POETs），这是一种基于模拟退火的算法，无需梯度信息。实现中修正了原始支配关系（从弱Pareto支配改为严格Pareto支配），通过增量更新方案将每次迭代的排序成本从O(n²m)降低到O(nm)，并添加多链并行执行以提高前沿覆盖。

Result: 在无细胞基因表达模型和血液凝固级联模型上进行了演示。合成数据研究显示参数可识别性结构，个体速率常数偏差数倍但模型预测准确度达7%。五重复覆盖分析确认时序特征可靠覆盖而峰值振幅系统性地过于自信。验证显示集成预测能保持在10%误差范围内。

Conclusion: ParetoEnsembles.jl通过使集成生成轻量化和易于访问，旨在降低机制建模中常规不确定性表征的门槛。

Abstract: Mathematical models of natural and man-made systems often have many adjustable parameters that must be estimated from multiple, potentially conflicting datasets. Rather than reporting a single best-fit parameter vector, it is often more informative to generate an ensemble of parameter sets that collectively map out the trade-offs among competing objectives. This paper presents ParetoEnsembles.jl, an open-source Julia package that generates such ensembles using Pareto Optimal Ensemble Techniques (POETs), a simulated-annealing-based algorithm that requires no gradient information. The implementation corrects the original dominance relation from weak to strict Pareto dominance, reduces the per-iteration ranking cost from $O(n^2 m)$ to $O(nm)$ through an incremental update scheme, and adds multi-chain parallel execution for improved front coverage. We demonstrate the package on a cell-free gene expression model fitted to experimental data and a blood coagulation cascade model with ten estimated rate constants and three objectives. A controlled synthetic-data study reveals parameter identifiability structure, with individual rate constants off by several-fold yet model predictions accurate to 7%. A five-replicate coverage analysis confirms that timing features are reliably covered while peak amplitude is systematically overconfident. Validation against published experimental thrombin generation data demonstrates that the ensemble predicts held-out conditions to within 10% despite inherent model approximation error. By making ensemble generation lightweight and accessible, ParetoEnsembles.jl aims to lower the barrier to routine uncertainty characterization in mechanistic modeling.

<details>
  <summary>Details</summary>

Method: 开发数学模型整合确定性成本和收入损失，优化群体检测算法池大小，使用德国社会经济面板收入数据，通过混合蒙特卡洛实验评估COVID-19大流行期间的经济成本。

Result: 蒙特卡洛实验显示：考虑收入损失时，最优群体检测算法选择发生显著变化；仅考虑确定性成本会系统性地低估总经济成本；隔离时间较短的算法更具吸引力。

### [6] [Quantifying Cross-Modal Interactions in Multimodal Glioma Survival Prediction via InterSHAP: Evidence for Additive Signal Integration](https://arxiv.org/abs/2603.29977)

Main category: cs.LG

<details>
  <summary>Details</summary>
Motivation: 多模态深度学习在癌症预后中通常假设能从跨模态协同交互中获益，但这一假设在生存预测环境中尚未得到直接验证。本研究旨在量化胶质瘤生存预测中的跨模态交互作用。
