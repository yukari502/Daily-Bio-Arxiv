### [1] [AlphaFunctor: Bridging The Gap Between Protein Function Annotation and Property Prediction](https://arxiv.org/abs/2607.12196)
*Xiang Liu,Anna E. Yee,Josh V. Vermaas,Daniel R. Woldring,Guo-Wei Wei*

Main category: q-bio.BM

TL;DR: AlphaFunctor is a category theory-based platform that predicts protein functions (Gene Ontology terms) directly from sequence, then maps those functions to downstream property predictions using topological and neural network methods, achieving state-of-the-art performance without task-specific redesign.


<details>
  <summary>Details</summary>
Motivation: Current protein function and property prediction methods are designed for specific datasets and specific property/function subsets, creating a large gap between function annotation and property prediction. The authors aim to bridge this gap with a unified approach.

Method: AlphaFunctor uses category theory as its foundation. It predicts Gene Ontology terms directly from protein sequence, then uses topological spectral theory, path-complex neural networks, and protein domain analysis to map these function predictions onto downstream property predictions. It is (pre)trained on ~0.6 million protein function data points.

Result: AlphaFunctor achieves state-of-the-art protein function annotation on three benchmark datasets. It also outperforms dataset-specific and task-specific competing predictors when mapping qualitative function annotations to various qualitative and quantitative property predictions without task-specific network redesign.

Conclusion: AlphaFunctor demonstrates that protein functions and properties can be directly derived from sequence, and that a category theory-based foundation model can effectively bridge function annotation and property prediction across diverse tasks without requiring task-specific redesign.

Abstract: The fundamental relationship among protein sequence, structure, function, and physicochemical properties is a central principle in biology. While in principle protein function and properties should be able to be derived directly from protein sequence, in practice protein function and property prediction methods have been designed around specific datasets and specific property or function subsets, leading to an enormous gap between function annotation and property prediction. To address these challenges, we introduce AlphaFunctor, a category theory based foundation model-like platform to bridge the gap between protein function annotation and property prediction. Based on the hypothesis that protein function and properties can be directly derived from protein sequence, AlphaFunctor predicts protein functions as represented by Gene Ontology terms directly from sequence. Using these function predictions, AlphaFunctor further maps protein functions using topological spectral theory, path-complex neural networks, and protein domain analysis onto downstream property prediction. AlphaFunctor is (pre)trained in nearly 0.6 million protein function data points to deliver the state-of-the-art protein function annotation on three benchmark datasets. Without task-specific network redesign, AlphaFunctor maps qualitative protein function annotation to various qualitative and quantitative protein property predictions, outperforming other dataset-specific and task-specific competing predictors.

### [2] [Contrasting statistical patterns in melodic and molecular evolution reveal distinctive constraints in a culturally evolving system](https://arxiv.org/abs/2607.12673)
*John M McBride,W Tecumseh Fitch*

Main category: q-bio.PE

TL;DR: This paper adapts bioinformatics alignment methods for rhythm-aware analysis of 40,000 Irish folk melodies, revealing cultural evolution patterns distinct from molecular evolution through mutability, substitution matrices, positional conservation, and covariance analyses.


<details>
  <summary>Details</summary>
Motivation: To overcome the challenge that melodies encode rhythm (which breaks standard sequence-alignment assumptions) and to enable large-scale automated melodic alignment for studying cultural evolution using bioinformatics tools.

Method: Developed a rhythm-aware alignment method and applied it to 40,000 Irish dance tune variants. Used four canonical bioinformatics analyses: mutability, substitution matrices, positional conservation, and covariance.

Result: The rhythm-aware alignment enabled the first large-scale automated melodic alignment. The four bioinformatics analyses revealed patterns distinct from molecular evolution, showing different forces shape each domain (biochemical/biophysical for proteins; memory, motor, and social biases for melodies).

Conclusion: Bioinformatics provides a powerful conceptual and algorithmic framework for studying cultural evolution, enabling large-scale analysis of music transmission that has been discussed for centuries but not previously analyzed at scale.

Abstract: Evolved sequences can be used to infer the rules of evolution. Orally transmitted folk melodies are evolved sequences whose similarity to protein sequences (one-dimensional, drawn from a limited alphabet) invites application of bioinformatics methods to study cultural evolution. A major obstacle is that melodies encode rhythm, which breaks some assumptions of standard sequence-alignment algorithms. We develop a rhythm-aware alignment method and apply it to \num{40000} Irish dance tune variants, enabling the first large-scale automated melodic alignment. Four canonical bioinformatics analyses -- mutability, substitution matrices, positional conservation, and covariance -- reveal patterns distinct from those of molecular evolution, revealing the forces that shape each domain: biochemical and biophysical constraints for proteins; memory, motor, and social biases for melodies. Together the results show that bioinformatics provides a powerful framework -- conceptual as much as algorithmic -- for studying cultural evolution. Although the cultural transmission of music has been discussed for centuries, here we show how to analyze it at large scale.

### [3] [On Tree-Network Distinguishability and Full Identifiability of Phylogenetic Networks](https://arxiv.org/abs/2607.12919)
*Jari Brits,Niels Holtgrefe,Leo van Iersel,Samuel Martin*

Main category: q-bio.PE

TL;DR: This paper proves that the semi-directed topology of level-1 phylogenetic networks is fully identifiable under JC, K2P, and K3P models on a biologically reasonable parameter space, and that phylogenetic networks can be distinguished from trees (unless the network is a tree or contains 2-blobs) under JC and K2P.


<details>
  <summary>Details</summary>
Motivation: Phylogenetic networks are essential for modeling reticulate evolutionary events (recombination, HGT, hybridization), but it was unclear whether their topology could be uniquely determined from sequence data under standard substitution models. Prior work only established generic identifiability (off a measure-zero set), leaving open whether identifiability holds everywhere.

Method: The authors analyze the identifiability of phylogenetic network topology under Jukes-Cantor, Kimura 2-parameter, and Kimura 3-parameter models. They examine the mapping from network parameters to leaf-pattern distributions, proving full identifiability on a parameter space where substitution rates are probabilistic and mixing parameters are not 0 or 1. They also compare distributions induced by networks vs. trees.

Result: (1) The semi-directed network parameter of a level-1 phylogenetic network (modulo redirecting triangles) is fully identifiable under JC, K2P, and K3P at every point of the parameter space, not just off a measure-zero set. (2) Under JC and K2P, no phylogenetic network and tree can produce the same leaf-pattern distribution unless the network is actually a tree or a tree augmented with 2-blobs.

Conclusion: Reticulate evolution generally leaves a detectable signature in leaf-pattern distributions under standard substitution models. These results have broader implications for identifiability in other models, including coalescent-based models, and provide strong theoretical guarantees for inferring phylogenetic networks from sequence data.

Abstract: Phylogenetic networks generalize phylogenetic trees to evolutionary histories that include reticulate events such as recombination, horizontal gene transfer, and hybridization. Under a Markov model of nucleotide substitution, a phylogenetic network determines a distribution of leaf-patterns. Here, we study the identifiability of the network topology from this distribution under the Jukes-Cantor (JC), Kimura 2-parameter (K2P), and Kimura 3-parameter (K3P) models. Our first result is that the semi-directed network parameter of a level-1 phylogenetic network (modulo redirecting triangles) is fully identifiable under all three models, on a biologically reasonable parameter space in which substitution rates are probabilistic and mixing parameters are non-trivial (i.e., not 0 or 1). In contrast to the generic identifiability established in prior work, this holds at every point of the parameter space, not merely off of a measure-zero subset. Our second result distinguishes phylogenetic networks from phylogenetic trees, on the same parameter space, under JC and K2P. We prove that no phylogenetic network and phylogenetic tree can induce the same leaf-pattern distribution unless the network is a tree, possibly augmented with certain substructures called $2$-blobs. This means the presence of reticulate evolution creates, in most cases, a detectable signature in the leaf-pattern distribution. More broadly, these results have consequences for identifiability beyond the models and network classes studied here, including for several coalescent-based models.

### [4] [Real-time fall detection based on vision for low-power edge platforms](https://arxiv.org/abs/2607.12909)
*Wenjun Xia,Zhicheng Peng,Haopeng Li,Zhengdi Zhang*

Main category: q-bio.NC

TL;DR: A physics-informed fall detection framework using dual Liquid Time-Constant (LTC) neural networks to model fall as a stability-loss event, achieving sub-50K parameters with competitive accuracy and physical interpretability for edge deployment.


<details>
  <summary>Details</summary>
Motivation: Existing vision-based fall detection methods treat falling as static pose classification or discrete temporal pattern matching, ignoring the fundamental instability dynamics of the human support system. This oversight limits both accuracy and physical interpretability.

Method: A dual-LTC architecture with Center-of-Mass (CoM) and Base-of-Support (BoS) subsystems connected via a learnable coupling module, plus a Stability Manifold classifier using Lyapunov-inspired metrics for boundary detection, and counterfactual trajectory projection with TTC estimation for early warning.

Result: Validated on a two-class dataset (Normal vs. Falling); achieves competitive accuracy with sub-50K parameters, enabling real-time inference on resource-constrained edge devices.

Conclusion: Physics-informed modeling of falling as a dynamical stability-loss event outperforms conventional CNN-RNN pipelines in physical interpretability and efficiency, with future work extending to three-state (Normal, Falling, Fallen) temporal transition prediction.

Abstract: Falling detection is vital for elderly care and intelligent surveillance; however, prevailing vision-based approaches predominantly frame it as static pose classification or discrete temporal pattern matching, fundamentally overlooking the instability dynamics of the human support system. This paper proposes a physics-informed falling detection framework that recasts falling as a stability-loss event in a coupled dynamical system. We introduce a novel dual-LTC architecture comprising a Center-of-Mass (CoM) subsystem and a Base-of-Support (BoS) subsystem, both instantiated as Liquid Time-Constant (LTC) neural networks to continuously model inertial trajectory evolution and ground-contact adjustment through adaptive time constants, Physical interpretability of falling motion. A learnable coupling module emulates physical interaction between the two subsystems, while a Stability Manifold classifier operates in the joint latent space to detect boundary crossing via Lyapunov-inspired stability metrics. Complementary counterfactual trajectory projection and Time-to-Collision (TTC) estimation further enable irreversibility assessment and early warning. The architecture is designed to support a three-state prediction paradigm (Normal, Falling, Fallen); in this preliminary study, we validate the core stability discrimination capability on a two-class dataset (Normal vs. Falling), leaving the full three-state temporal transition to future work. Unlike conventional CNN--RNN pipelines, the proposed formulation encodes continuous-time mechanical inertia, yielding a sub-50K-parameter network capable of real-time inference on resource-constrained edge devices. Extensive experiments demonstrate competitive accuracy with superior physical interpretability, validating its efficacy for low-compute visual fall detection.

### [5] [Optimal photostimulation selection for iterative activity maps](https://arxiv.org/abs/2607.12930)
*Jacob J. Morra,Kaitlyn E. Fouke,Owen Traubert,Eva A. Naumann*

Main category: q-bio.NC

TL;DR: OPhELIA is a Bayesian framework for trial-efficient causal connectivity mapping in all-optical optogenetics, achieving near-exhaustive connectome recovery with only 5% of trials using active learning and compressed sensing.


<details>
  <summary>Details</summary>
Motivation: Exhaustive connectivity mapping in all-optical holographic optogenetics is experimentally prohibitive due to combinatorial complexity, tissue heating, photodamage, and time constraints.

Method: OPhELIA combines Beta-Bernoulli connectivity inference with an ambiguity-based acquisition heuristic and learned priors from pre-stimulation neural activity, augmenting active learning and compressed sensing.

Result: OPhELIA with active learning improves trial-efficient approximation of exhaustive functional connectomes; with compressed sensing, it most closely recovers an exhaustive connectome using only 5% of trials.

Conclusion: OPhELIA is a sample-efficient framework for causal connectomics, enabling practical large-scale circuit mapping.

Abstract: All-optical two-photon holographic optogenetics enables causal circuit mapping by stimulating defined neurons or ensembles while imaging population activity. Yet exhaustive connectivity mapping remains experimentally prohibitive because of combinatorial complexity, tissue heating, photodamage, and experimental time. We present OPhELIA (Optimal Photostimulation sElection for Iterative Activity maps), a Bayesian framework for selecting informative perturbations under limited trial budgets. OPhELIA combines Beta-Bernoulli connectivity inference with an ambiguity-based acquisition heuristic and learned priors derived from pre-stimulation neural activity, augmenting active learning and compressed sensing. In standalone simulations and in vivo larval zebrafish visuomotor experiments, OPhELIA with active learning improves trial-efficient approximation of exhaustive functional connectomes. In combinatorial in vivo experiments, OPhELIA with compressed sensing most closely recovers an exhaustive connectome using only 5% of trials. These results establish OPhELIA as a sample-efficient framework for causal connectomics.

</details>
